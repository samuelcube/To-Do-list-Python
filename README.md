# 📝 Python Todo App (JSON)

Eine einfache Todo-Listen-App in Python mit Speicherung in JSON.

---

## 🚀 Features

- ➕ Todo hinzufügen
- 🗑️ Todo löschen (per Nummer)
- 📋 Todos anzeigen
- 💾 Automatisches Speichern in JSON-Datei
- 📂 Laden der Daten beim Start
- 🛡️ Fehlerbehandlung (kein Crash bei falscher Eingabe)

---

## 📦 Technologien

- Python 3
- JSON (für Datenspeicherung)

---

## ▶️ Starten

1. Repository klonen oder Datei herunterladen
2. Python Datei ausführen:

```bash
python main.py
